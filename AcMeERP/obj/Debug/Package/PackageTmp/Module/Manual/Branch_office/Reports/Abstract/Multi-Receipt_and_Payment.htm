Multi Abstract Receipt and Payments

Multi abstract receipt and payments shows the abstracts of all the months for the specified period. The receipts and  payments for the specified date range are displayed monthwise in a single report.

To generate multi abstract receipt and payments:

  1. Click Reports > Abstract > Multi - Receipt And Payments.
    The Multi Abstract (Receipt and Payments) customise dialog box is displayed. View image

  2. Select the options in the customise dialog box.

  3. Click Ok.
    The Multi Abstract (Receipt and Payments) report is displayed. View image

  • If the date range ( Date From and date To ) is not selected, the report is displayed for all the months of an active Accounting Period.
