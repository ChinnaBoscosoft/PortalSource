Manage Branch Office

Import Branch Office Details from Excel

This feature helps the head office admin to download the excel file from the portal and the excel file contains all the fields to be entered. The head office admin can enter all the details and upload it again and based on the details entered in excel the branch offices are created.

To import branch office from excel:  

  1. Click Tools > Import Branches.
    The Import Branches page is displayed. View image

  2. Click in the page to download excel file.
    The excel file is downloaded.

  • The excel file contains all the fields to create branch offices. The user must enter all the details of the branch in the excel file and upload the filled excel sheet.

  1. Enter the details in the excel file.

  2. Click Browse to select the file from the system.
    The File Upload dialog box is displayed. View image

  3. Select the excel file from the system.

  4. Click Open.

  5. Click Upload.
    The summary of the uploaded details is displayed.

  • The summary of the details uploaded is displayed in the summary area of the page. This shows the number of records uploaded and number of records skipped.
