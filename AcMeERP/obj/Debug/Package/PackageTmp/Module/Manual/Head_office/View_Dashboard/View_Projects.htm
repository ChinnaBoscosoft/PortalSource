View Projects

The currently running projects in the branch office are listed branchwise and the details like opening balance, Receipt, payment and closing balance for each project is displayed.

To view projects:

  1. Click Home.
    The dashboard is displayed. View image

  2. Select Branch from the list. To select all the branches select All from the list.

  3. Select Date from the calendar box.
    The available project under the branch is displayed. View image
