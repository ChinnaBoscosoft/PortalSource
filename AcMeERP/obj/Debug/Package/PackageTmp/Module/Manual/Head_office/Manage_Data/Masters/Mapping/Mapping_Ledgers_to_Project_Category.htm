Mapping Ledgers to Project Category

The ledgers are useful for the project for making transactions. When these ledgers are mapped with project category, all the projects that are created under the particular project category uses same ledgers. As all the projects under a category uses common ledgers mapped to it helps in generating consolidated and  comparative reports.

To map ledgers to project category:  

  1. Click Data > Masters > Map Ledgers
    The Ledger Mapping page is displayed. View image

  2. Select Project Category from the list.

  3. Select the ledgers by selecting the check boxes in the page.

  4. Click Save.
