Manage Users

Activated and Deactivate users

You can deactivate, or disable, a user is not deleted instead his/her details remain in the portal. The deactivated users cannot logon to the portal.

To deactivate user:

  1. Click Users > User.
    The User view page is displayed.  View image 

  2. Click (Deactivate) under status column in User View page to deactivate the user. The Are you sure to deactivate? message is displayed.
    Click Ok.
    The icon under status column changes to (Activate).

To activate user:

  1. Click Users > User.
    The User view page is displayed. View image

  2. Click (Activate) under status column in User View page to activate the user.
    The Are you sure to activate? message is displayed.
    Click Ok.
    The icon under status column changes to (Deactivate).
