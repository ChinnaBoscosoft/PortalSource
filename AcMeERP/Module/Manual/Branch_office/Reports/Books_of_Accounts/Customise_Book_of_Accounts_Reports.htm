Book of Accounts

Customise Book of Accounts Reports

You can customize the report based on the requirement specified in the customize dialog box and the report is displayed based on the options selected.

The following are the properties that can be set for each report.

  • Date

    Date range (Date From and Date To) :

    Select the date range from the date boxes. The data will be displayed based on the date range.

    Include Narration

    Select include narration to show the narration in the report.

    Show daily Balance

    Select show daily balance to show the daily cash and bank balances.

    Include Detailed Balance

    Select include detailed balance to shows detailed balances of cash in hand and bank accounts.

  • Branch

    The list of branch offices are listed in Branch tab.  

     

  • Legal entity

    The list of legal entities are listed under Legal Entity tab.

  • Project

    Project

    The Projects are listed based on the branch(s) and the legal entity(s) selected.

  • Ledger

    Ledger Group

    Ledger groups mapped to the projects are displayed.

    Ledger

    Based on the ledger group selected the ledgers are listed. Select the particular ledgers to their transaction history.  

  • Reports Setup

    Show Titles at Each page

    Select show titles at each page, the institute name and report name are displayed on all the pages of the report. If the Show Titles At Each Page check box is not selected only the first page of the report will have the institute name and report name.

    Report Date

    The date on which the report is taken and it appears in the footer of every page of the report. It can be modified based on the you preference.

    Vertical Line

    Select vertical line check box to show vertical lines in the report, which differentiates the columns.

    Horizontal line

    Select horizontal line check box to show horizontal lines in the report, which differentiates the rows.

    Title Alignment

    Select the alignment of title in the report: Left alignment means the title is aligned along the left margin, Right alignment means the title is aligned along the right margin, Center alignment means the title is aligned to the center in the title area.  

    Show Report Logo

    Select show report logo to show the logo of the organisation is shown in the report.

    Show Page Number

    Page numbering is the process of giving sequence of numbers for the pages in the report. Select show page number to show the page number in the bottom left of the pages in the report.

 

  • The customisation can also be done by clicking the (Customise) in report page.
