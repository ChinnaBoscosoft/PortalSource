Getting Started

Common Operations

Click on the given icon to refresh the page.

Click on the given icon to deactivate user.

Click on the given icon to activate user.

Click on the icon to reset password of a user in the users page. The new password will be sent to the user through email. The password is sent to the email account given during user creation.

Click on the icon to edit a record. The record will be available for edit. Make the necessary changes and click Save.

Click on the icon to delete the record.

Click on the icon to view the details of a record.

Click on the icon, the login credentials are sent to the user via the mail given in communication details.

 

Select the file type   from the list to export the report. The file type include PDF, XLS, XlSX, RTF, MHT, HTML, TEXT, CSV and IMAGE.   

Click on the icon  to go back to the original  report during the time of drill down.

Click on the icon to export the data from the page to excel.
