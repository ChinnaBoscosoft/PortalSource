Manage Users

Role

The permissions to perform certain operations are assigned to specific roles. In Acme.erp Portal, the privileges are set by the branch office admin for branch office users.

To add user role:

  1. Click Users > Role.
    The User Role View page is displayed. View image

  2. Click Add User Role in the top right side of the User Role View page.
    The User Role (Add) page is displayed. View image

  3. Enter Role Name in the box.

  4. Click Save.
    The Role saved message is displayed.

  • The user roles like admin, branch admin, etc, exist in the system and they cant be edited. The user roles added newly can be edited. They are differentiated by (Edit) icon.
