Download Key

The key is an xml file which contains the license details for Acme.erp. If the license expires the user can request for the new license to the portal admin.

To download key:

  1. Click Tools > Download Key.
    The Download Key page is displayed. View image

  2. Click on to download the key for the branch.
    The download dialog is displayed. View image

  3. Click Save.

  4. Click Ok.
