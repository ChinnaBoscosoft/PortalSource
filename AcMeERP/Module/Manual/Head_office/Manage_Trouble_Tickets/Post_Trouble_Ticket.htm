Post Trouble Ticket

Reporting a problem to the head office admin  through ticketing is done using post trouble ticket. The replies for the ticket can be made by all the user in the branch office users, head office users and portal users. The trouble tickets are separated for each branch office and the

To post trouble ticket

  1. Click Trouble Ticketing > Post Ticket.
    The Tickets page is displayed. View image

  2. Click Post Ticket in the upper-right side of the page.
    The Post Ticket dialog box is displayed. View image

  3. Type Subject and Description in the boxes.

  4. Select Priority from the list.

  5. Click Save.
    The successfully saved message is displayed.
