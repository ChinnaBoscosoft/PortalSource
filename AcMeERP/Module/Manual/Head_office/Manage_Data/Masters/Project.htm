Project

Every non-profit organization may run Health Care Centres, Colleges, Schools, etc. In Acme.erp, these services are considered as Projects and only after creating projects the transactions can be entered. The projects that involve the local currency alone should be created under local division and Project that involves foreign currency should be created under foreign division

    To add project:

    1. Click Data > Masters > Project.
      The Project View Page is displayed. View image

    2. Click Add Project in the upper-right side of the Project View page.
      The Project (Add) page is displayed. View image

    3. Enter project code in the Code box.

    4. Select  Society Name and Project Category from the lists.

    5. Type project name in the Project box and description about the project in the Description box.

    6. Select Started On and Closed On from the calendar boxes.

    7. Select Division from the list.

    8. Enter address of the project in Address box.

    9. Click Save.
      The Project is Saved message is displayed.

    10. Click Ok.

    11. Click Close to go back to Project view page.
