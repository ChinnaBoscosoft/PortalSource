Mapping Project(s) to Branch Office

All the projects are created in the head office and these projects must be mapped to the branch office.

To map project(s) to branch office:

  1. Click  Data > Masters > Map Projects.
    The Project Mapping page is displayed. View image

  2. Select Branch from the list.

  3. Select the projects by selecting the check boxes in the page.

  4. Click Save.
    The Saved Successfully message is displayed.
