Ledger

Ledgers are masters created in head office  used to post transactions. A ledger records the classified and summarized financial information; and they hold the information that is required for totaling of revenues, calculating income and expenses, and preparing financial statements.

    To add ledgers:

    1. Click Data > Masters > Ledger.
      The Ledger View page is displayed. View image

    2. Click Add Ledger in the upper-right side of the Ledger View page.
      The Ledger (Add) page is displayed. View image

    3. Select Ledger Group from Under the Group list.

    4. Enter the code in the box.

    5. Select Ledger Type from the list.

    6. Type ledger name in the Name box and description in the Notes box.

    7. Select "Include Cost Centre" to include cost centre for this ledger and "Set this as Bank Interest Ledger" to set this as bank  interest ledger.

     

    • Include Cost Centre: This option is to select the cost centre when the cost center enabled ledger is selected while making transactions. If cost center is not included for a ledger the cost centre will not appear while making transactions.  

    • Set this as Bank Interest Ledger: This option is to make a ledger as bank interest ledger. These ledgers are mostly used in fixed deposits for the receiving interest.  

    1. Click Save.
