Import Masters from Excel

It is an option used instead of adding the masters one by one. This feature allows the head office user to download the template in excel, enter all the masters in the downloaded template and upload it back to the portal.  

To import masters from excel:

  1. Click Tools > Import Masters.
    The Import Masters page is displayed. View image

  2. Click in the page to download excel file.
    The excel file is downloaded.

  3. Enter the details in the excel file.

  4. Click Browse to select the file from the system.
    The File Upload dialog box is displayed. View image

  • Validate and check all the details in the excel.

  1. Select the excel file from the system.

  2. Click Open.

  3. Click Upload.
    The summary of the uploaded details is displayed.

  • The summary of the details uploaded is displayed in the summary area of the page. This shows the number of records uploaded and number of records skipped.
