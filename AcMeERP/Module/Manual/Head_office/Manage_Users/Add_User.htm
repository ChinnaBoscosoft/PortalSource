Manage Users

User

This feature allows the administrators to add users, create roles, set rights, reset password, activate a user and deactivate a user.  

To add users:

  1. Click User > user.
    The User View page is displayed. View image

  2. Click Create User button in the top right side of the User view page.
    The User Add page is displayed. View image

  3. Type First Name in the box.

  4. Select Role from the list.

  5. Type Username, Password, Confirm password, Mobile Number and Email in the boxes.

  6. Click Save.
    The User saved message is displayed.

  7. Click Ok.

  8. Click Close to go back to User View page.
