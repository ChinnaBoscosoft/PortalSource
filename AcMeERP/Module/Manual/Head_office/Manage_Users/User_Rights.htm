Manage Users

User Rights

User rights specifies the rights for each user role. The rights are set in the User rights page where, the admin grants user role permission to access only the needed pages.

To set user rights

  1. Click User > User Rights.
    The User Rights page is displayed. View image

  2. Select the Role from the left pane and select the check boxes in the left to allow the user role to access the feature.

  • The rights will be saved automatically as the user select the check boxes.

  • When the user is created, rights set for a particular role will be applied.
