Manage Users

Change Password

The user can anytime change the password using this feature; Password should have combination of minimum one character, one number and minimum length is 8 including special characters and numbers.

To change password:

  1. Click User > password.
    The Change Password page is displayed. View image

  2. Enter Old Password, New Password and Confirm New Password in the boxes.  

  3. Click Change.
    The Your password has been changed successfully message is displayed.
