Branch office

The branch offices are the subgroup of the head office. There can be any number of branch offices under a head office. 

To add branch office:

  1. Click Office > Branch Office.
    The View Branch Office page is displayed. View image

  2. Click Create Branch Office at the upper-right side of the  View Branch Office page.
    The Branch Office (Add) page is displayed. View image

  3. Select Head office under which the branch falls from the list.

  4. Type Code/Username and Name of the branch office in the boxes.

  5. Click Standalone or Client Server Deployment Model.

  6. Enter the name of the Person Incarge in the box.

  7. Type Name, Mobile No, Phone No, Email, Address and City in the boxes.

  8. Select Country and State/Province from the list.

  9. Enter Postal/Zip Code in the box.

  10. Click Save.
    The Branch Office is Saved message is displayed.

  11. Click Ok.

  12. Click Close to go back to Branch Office view page.
